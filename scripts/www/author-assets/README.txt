================================================================
作者卡片图片资源目录
================================================================

这个目录里放了三张图片，对应 "关于作者" 弹窗中的：

1. avatar.jpg          ← 作者头像（圆形展示，推荐正方形图片）
2. wechat-qrcode.svg   ← 微信二维码（替换为你自己的微信二维码图片）
3. reward-qrcode.svg   ← 打赏二维码（替换为你自己的打赏码图片）

================================================================
替换方法
================================================================

【方法一：保持文件名不变，直接覆盖（推荐）】
   - 把你的头像图片命名为  avatar.jpg    放到这个目录
   - 把你的微信二维码图片命名为  wechat-qrcode.jpg  放到这个目录
   - 把你的打赏码图片命名为  reward-qrcode.jpg  放到这个目录

   然后修改 texts.js 中 AUTHOR_ASSETS 对应的文件名后缀即可：

   var AUTHOR_ASSETS = {
     dir: 'author-assets',
     avatar: 'author-assets/avatar.jpg',               ← 改这里的文件名
     wechatQrcode: 'author-assets/wechat-qrcode.jpg',  ← 改这里的文件名
     rewardQrcode: 'author-assets/reward-qrcode.jpg'   ← 改这里的文件名
   };

   （.jpg / .png / .svg 都可以）

【方法二：修改 texts.js 中 AUTHOR_ASSETS 的路径】
   - 把图片放进这个目录，随意命名
   - 打开 texts.js，修改 AUTHOR_ASSETS 里对应路径即可
   - 修改后首页和扫码页会同步更新

================================================================
提示
================================================================
- 头像建议：正方形图片，宽度 200~400 像素，会自动圆形裁剪
- 二维码建议：正方形图片，宽度 200~400 像素，清晰可扫
- 修改完成后刷新页面（Ctrl+R / F5 / Cmd+R）即可看到效果
