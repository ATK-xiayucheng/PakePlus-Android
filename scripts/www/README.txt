▶ 宝塔面板部署步骤（最简单的方法）
============================================================

方法一：上传解压到网站根目录（推荐新手）

  1. 登录宝塔面板 → 点击左侧"网站"
  2. 点击"添加站点"
     - 域名：填写你要使用的域名（如 scan.example.com）
     - 根目录：默认即可（如 /www/wwwroot/scan.example.com）
     - FTP：不创建
     - 数据库：不创建（纯静态网页，不需要数据库）
     - PHP版本：纯静态（本项目不需要 PHP）
  3. 点击"提交"创建站点
  4. 点击刚创建的站点 → "根目录" → 进入文件管理器
  5. 删除默认的 index.html 文件
  6. 点击"上传" → 上传本压缩包 → 解压到当前目录
  7. 在浏览器访问你的域名 → ✅ 完成！

方法二：上传单个文件（更灵活）

  1. 同上步骤 1-5
  2. 把压缩包解压到你电脑
  3. 把所有文件（index.html、scan.html、renderer.js、
     styles.css、texts.js、author-assets/）上传到网站根目录
  4. 访问你的域名 → ✅ 完成！

▶ 使用说明
============================================================

1. 首页（项目管理）
   - 打开网站后会看到项目管理首页
   - 点击"新建项目"创建一个新的扫码项目
   - 填写型号、选择时间 → 确定后自动跳转到扫码页

2. 扫码页面
   - 点击"提交"或直接按回车提交序列号
   - 序列号会自动去重，重复时会弹窗提醒
   - 数据自动保存在当前浏览器中（不会上传到服务器）

3. 导出 Excel
   - 扫码记录点击"导出Excel"按钮即可下载

4. 作者信息
   - 页面底部有"关于作者/联系客服"按钮，点击查看

▶ 常见问题
============================================================

Q1: 数据会保存在服务器上吗？
A1: 不会。所有数据（序列号、项目信息等）全部保存在
    用户自己的浏览器 localStorage 中。

Q2: 换浏览器或清缓存后数据会不会丢失？
A2: 会。建议每天扫码完成后立即点击"导出Excel"
    保存到本地，以防数据丢失。

Q3: 能不能在手机浏览器上用？
A3: 可以。支持任何现代浏览器（Chrome/Edge/Firefox/Safari 等）。

Q4: 如何修改作者信息或界面文字？
A4: 修改 texts.js 文件。里面把所有可见的文字都
    用变量集中管理，修改这一个文件即可。

Q5: 如何替换作者头像、微信二维码、打赏码？
A5: 把你的图片命名为以下文件名，放到 author-assets/ 目录：
    - avatar.jpg           ← 作者头像（建议正方形，200x200 像素以上）
    - wechat-qrcode.jpg    ← 微信二维码
    - reward-qrcode.jpg    ← 打赏码
    （.jpg/.png/.svg 都可以）
    然后修改 texts.js 中 AUTHOR_ASSETS 部分的路径名。
